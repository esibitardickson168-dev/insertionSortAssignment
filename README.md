        ## the program Overview
  
This program implements the **Insertion Sort algorithm in C** with an interactive dataset option.  
The program allows the user to either Manually enter data  OR Automatically generate random data  

It then sorts the array using insertion sort and displays a small sample of the sorted result alongside with the execution time.

         ?> its Features >?
- Interactive choice for *manual or automatic dataset*
- Handles **up to 10,000,000 elements**
- Prints **first 20 elements** to verify sorting
- Measures **time taken** for sorting
How the loops work
 -for loop picks each element from the unsorted part to insert and While loop shifts larger sorted elements to the right to make space for the key.

  
       ?<  How the Program Works ?<

1. User enters the number of elements (`n`) — max 10,000,000.  
2. User chooses:
   - `1` → manual input  
   - `2` → automatic generation  
3. Program displays a **sample of the unsorted array**.  
4. Insertion Sort is applied.  
5. Program displays a **sample of the sorted array** and **time taken to sort the array**.
